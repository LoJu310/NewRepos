pwd - текущая директория(папка)
ls - вывод всех элементов в папке
cd - перход в эту папку
cd .. - выйти из папки
cat - просмотр содержимого файла
wc - вывод кол-ва строк,слов,символов в файле
mkdir имя папки - создание папки
touch имя файла - создание файла
nano имя файла - изменение содерждимого, ctrl+O - сохранить,enter, 
ctrl + X - вернуться назад
mv старое_имя новое_имя - переименование
rm имя файла - удалить файла
rm -r имя папки - удалить папку
